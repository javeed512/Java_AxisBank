
Lombok Project:
----------------
1. Download  project lombok.jar from https://projectlombok.org/download
2. run lombok.jar file using
	-> java -jar  lombok.jar
  Window will appear
3. click the specify location button and select the path for eclips/sts folder in which( .exe file )
4. install button (lombok installed successfully)
5. Restart IDE eclipse/sts.
